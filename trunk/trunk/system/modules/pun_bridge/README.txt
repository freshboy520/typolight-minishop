PunBB forum bridge
------------------

1. Set up the configuration file

Open file config/config.php and enter your punBB configuration details
as they are stored in the punBB configuration file (config.php).

2. Set up mod_rewrite

Rename file ._htaccess to .htaccess and move it into the root folder
of your punBB installation. Open the file in a text editor and adjust
the rewrite rules. Follow the instructions provided.
